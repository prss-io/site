 Windows Installation

 The installation for Windows is as follows:

  • Open the file called "PRSS-win32.exe" to install PRSS.
  • If prompted by Windows, choose "More Info" and "Run anyway". PRSS releases are digitally signed and secure.
  •   Make sure to set up Git if you plan on deploying to Github Pages with PRSS.
  • You are ready to roll! Please share your feedback through the in-app "Help & Support" section.
