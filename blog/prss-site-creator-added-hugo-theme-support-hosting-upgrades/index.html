Hugo Theme Support & Hosting Upgrades

Published on 13 Sep, 2025
Photo by stein egil liland via Pexels

We're excited to announce that we've added (Beta) support for select Hugo themes in the latest version of PRSS Site Creator. This means you'll now have even more options when choosing a theme for your website!

Stay tuned for a hand-picked selection of Hugo themes in the in-app Theme Manager. Here's an example of the "Terminal" Hugo theme:

How does Hugo theme support work?

We have added a new "Hugo" handler to accompany our longstanding "React" handler. What the Hugo handler does is to convert posts into Markdown format for the Hugo executable to process. The configuration that Hugo uses is also customized to match the output that PRSS sites expect. This ensures that links, images, and pagination are in perfect working order.
