 MacOS Installation

 The installation for OS X is as follows:

  • Open the file called "PRSS-darwin.dmg".
  • A new window should open. Drag the PRSS icon shown to the Applications folder.
  • You can now search and open the PRSS app.
  • Non-store versions: If prompted by MacOS, choose "Open". PRSS releases are built with trusted open source components and are secure.
  • You are ready to roll! Please share your feedback through the in-app "Help & Support" section.
